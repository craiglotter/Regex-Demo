About The Author

Jan Goyvaerts is Chief Software Designer at his own software company JGsoft - Just Great Software. JGsoft develops and publishes various software packages including PowerGREP, RegexBuddy and EditPad Pro.

PowerGREP is a knowledge worker's Swiss army knife for searching through, filtering and retrieving information from piles of files using regular expressions. RegexBuddy is your perfect assistant for working with regular expressions, making it easy to create, test and save regexes. EditPad Pro is a popular text editor with particularly strong support for regular expressions.
